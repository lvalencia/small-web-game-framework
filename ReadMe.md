# Mica

A Small Code Generation Framework for ThreeJS Based Games

## Install

`npm install -g ./`

## Try out Generation

`mica new mygame`

Should generate and create the following project structure

```
mygame
├── index.html
├── package.json
├── public
│   └── js
│       ├── dependencies
│       │   ├── dat.gui.mjs
│       │   ├── stats.mjs
│       │   └── three.mjs
│       ├── helpers
│       │   └── axisGridHelper.mjs
│       └── main.mjs
└── server.js

4 directories, 8 files
```
